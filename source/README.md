# Setup

__VSCode__
* run VSCode in this directory: run `code .`
* open Julia REPL (`Alt+J`)
* press `]` in Julia REPL
* run `activate .`
* run `instantiate`
* run `precompile`
* press `backspace` or `Ctrl+C`
* you should be back in Julia REPL 

# Run
* run in Julia REPL `include("scripts/h-model-test.jl")`
